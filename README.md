# simplecountdown
A simple countdown

# Example
Add div container:
<pre><code>
<div id="myCountdown"></div>
</code></pre>

Add js to load countdown:
<pre><code>
<script type="text/javascript">
  var deadline = '2016-08-15 15:00:00';
  SimpleCountdown.autoDisplay('myCountdown', deadline);
</script>
</code></pre>
